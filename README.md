# My implementation of some acoustic echo cancellation filters

## Filters implemented
- Time domain filters
   - Least Mean Squares
   - Normalized Least Mean Squares
   - Block Least Mean Squares
   - Normalized Block Least Mean Squares
   - Kalman filter
- Frequency domain filters
   - FDAF
### List of filters is updating

## Technologies used:
- Python 3.10
- Torch 2.0
- Scipy
